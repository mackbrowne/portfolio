Code

Here are a collection of projects that I have worked on.

ClearConcert

Development and Maintenance - 2013 - ClearBlade

IBM Rational Team Concert has gone mobile! Use ClearConcert to check and update your work items from your Android or Apple device. I was involved in the initial development and then head of additional development and maintenance. Built with Phonegap and AngularJS

Safety Audit

Head of Development - 2013 - ClearBlade and Hatch

ClearBlade was hired by Hatch to build a safety audit tablet app that could be used by engineers in the field. This app created a better way to record data in the field. The engineer can create reports, attach photos and submit everything directly to a secure database back at the office. Built with Worklight, AngularJS and Jquery Mobile.

Worklight Social

Prototype Development - 2012 - ClearBlade

This was a fairly simple message board app that was designed to shocase the development stack using IBM Worklight, Websphere Liberty Application Server, Apache Derby, AngularJS and Jquery Mobile. The app was capable of facebook login with OAuth, posting, comments, tagging and geolocation.

MackSynth

Synthesizer Development - 2012 - Independent

This was a project for a class that I took to the next level. Using the riffWave.js library, the MackSynth can play a variety of digital waves in a 3 musical octave range. Each wave equation needed to be re arranged to output a frequency in Hz. I derived these custom equations myself then the riffwave tool would convert the wave data into audio.

Web Development - 2012 - Independent

Concurrent Web Crawling Using Clojure

Web Crawling - 2012 - Dr Ken Pu, UOIT

A web crawler written in Clojure that takes advantage of multi processor systems. The thesis studied the performance gains from Clojure's parallelism features. To do this, we collected screenshots from over 100 live traffic feeds using a separate thread to download from each camera.
